## Setup

```sh
# Install dependencies
yarn
```

```sh
# build package
yarn library build

# publish the library
yarn publish
```

### Getting started :octocat:

#### Prequalities
```
Node >= 12.9
yarn >= 1.22
```

After the setup run the example locally using

```sh
yarn dapp start
```
